placement menu always below

how the menu anchors

long-press a card (500ms hold). it lifts (scale 1.03 + shadow) over a dimmed, frosted-blur backdrop, and the menu always opens directly below it (10px gap). when the card sits too low for the menu to fit, the card itself slides up so the whole stack fits on screen — the menu never flips. the card stops at the top safe area (never off-screen); if even then the menu is taller than the screen, the menu scrolls. (blur is long-press only — the overflow menu below has no backdrop.)

interaction recap.
