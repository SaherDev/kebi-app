01 — colors canvas

canvas

three layers of warm paper: background, surface (cards, pills), and surface-2 (dividers, nested). Light mode is candlelit paper. Dark mode is warm near-black, never pure #000.

surface --surface #F1F1EF / #1C1A17
surface-2 --surface-2 #E8E8E5 / #26231F
02 — colors text

text

three text weights based on hierarchy. text is the primary copy. muted is meta, sublabels, secondary info. soft is timestamps, eyebrow labels, decorative.

text-muted --text-muted #8A8A87 / #9A938A
text-soft --text-soft #B5B5B1 / #6E6862

15 — iconography feather style

icons

stroke-based, 1.8px stroke width, square line caps. 18px in row icons, 14px in chips, 12px in toasts, 36px for the mascot avatar inside the AI button. Always inherits color via stroke: var(--text) etc.

check
success states, completed steps, confirmations
chevron
row navigation, expandable sections, place cards
plus
add a place, add to a group, expand options
close
dismiss, cancel, "not it" reactions
search
expandable search input in top pill
share-in
save trigger in top pill. tray + arrow dropping in. matches "drop it in" voice.
