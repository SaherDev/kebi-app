### RN-specific implementation notes (gaps between HTML and React Native)

These are things that work trivially in CSS but need explicit wiring in RN. Each must be verified before the track step is marked done.

**A2 â€” Tokens:**
- CSS custom properties â†’ NativeWind theme config + StyleSheet constants object. No inline hex values anywhere.
- `.dark` class toggle â†’ `useColorScheme()` + NativeWind dark variant (`dark:bg-surface`). Single source; components stay theme-agnostic.
- Inter font must be loaded via `expo-font` / `@expo-google-fonts/inter` before first render (part of splash).

**A3 â€” Navigation:**
- No tab bar. Expo Router file-based routing for screen stack only.
